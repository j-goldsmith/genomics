Phylos Bioscience Data

Phylos has made all of its existing DNA sequence data public, except in the cases where customers chose to keep their data private. The raw data itself is hosted on the NIH GenBank site, where it is easily searchable and accessible to researchers. Links to the data for each sample are below. Instructions and tools for downloading the raw data can be found here.

Full Genome Assembly:

A draft assembly of the genome of Cannabis sativa variety Full Genome Assembly: Cannatonic, generated by long-read PacBio sequencing and assembled using the Falcon assembler. This assembly has 11,110 contigs and an N50 of 129 kb. We continue to work to improve this assembly Short Targeted sequencing data: using various methods, and will release further updates in the future.

Full Genome Link
https://www.ncbi.nlm.nih.gov/nuccore/MNPR01000000

Targeted sequencing data:

Short-read sequence data from samples collected as part of the Cannabis Evolution Project and the Phylos Genotype test. This data was generated using extremely high-depth targeted amplicon sequencing at 3000 loci, on an Illumina NextSeq sequencer, and has an average experimental error rate of 0.3%. All of these samples are located on the Phylos Galaxy, and can be identified using the Phylos ID#. This data set will be continually updated as new samples are sequenced.
